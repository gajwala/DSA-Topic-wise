# 12. 3Sum

**LeetCode Link**: [15. 3Sum](https://leetcode.com/problems/3sum/)

**Difficulty**: Medium

**Topics**: Array, Two Pointers, Sorting

---

## Problem Statement

Given an integer array `nums`, return all the triplets `[nums[i], nums[j], nums[k]]` such that `i != j`, `i != k`, and `j != k`, and `nums[i] + nums[j] + nums[k] == 0`.

Notice that the solution set must not contain duplicate triplets.

### Examples

**Example 1:**
```
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
```

**Example 2:**
```
Input: nums = [0,1,1]
Output: []
```

**Example 3:**
```
Input: nums = [0,0,0]
Output: [[0,0,0]]
```

### Constraints
- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

---

## Approach 1: Brute Force

```javascript
function threeSum(nums) {
    const result = [];
    const n = nums.length;
    
    for (let i = 0; i < n; i++) {
        for (let j = i + 1; j < n; j++) {
            for (let k = j + 1; k < n; k++) {
                if (nums[i] + nums[j] + nums[k] === 0) {
                    result.push([nums[i], nums[j], nums[k]].sort((a,b) => a-b));
                }
            }
        }
    }
    
    // Remove duplicates
    return Array.from(new Set(result.map(JSON.stringify))).map(JSON.parse);
}
```

**Complexity**: O(n³) - Too slow!

## Approach 2: Hash Set

```javascript
function threeSum(nums) {
    const result = [];
    const n = nums.length;
    
    for (let i = 0; i < n; i++) {
        const seen = new Set();
        for (let j = i + 1; j < n; j++) {
            const complement = -(nums[i] + nums[j]);
            if (seen.has(complement)) {
                result.push([nums[i], nums[j], complement].sort((a,b) => a-b));
            }
            seen.add(nums[j]);
        }
    }
    
    // Remove duplicates
    return Array.from(new Set(result.map(JSON.stringify))).map(JSON.parse);
}
```

**Complexity**: O(n²) time, O(n) space - Better but duplicate handling is messy!

## Approach 3: Sort + Two Pointers (Optimal!) ✅

### Intuition
1. Sort the array
2. Fix one element, use two pointers for remaining two
3. Skip duplicates to avoid duplicate triplets

### Algorithm
1. Sort array
2. For each element i:
   - If nums[i] > 0, break (all remaining positive)
   - Skip duplicates of i
   - Use two pointers (left = i+1, right = n-1)
   - Find pairs that sum to -nums[i]
   - Skip duplicates of left and right

### Implementation

```javascript
/**
 * Sort + Two Pointers - Optimal
 * @param {number[]} nums
 * @return {number[][]}
 */
function threeSum(nums) {
    const result = [];
    nums.sort((a, b) => a - b); // Sort array
    
    for (let i = 0; i < nums.length - 2; i++) {
        // Skip positive numbers (all remaining will be positive)
        if (nums[i] > 0) break;
        
        // Skip duplicates for i
        if (i > 0 && nums[i] === nums[i - 1]) continue;
        
        let left = i + 1;
        let right = nums.length - 1;
        
        while (left < right) {
            const sum = nums[i] + nums[left] + nums[right];
            
            if (sum === 0) {
                result.push([nums[i], nums[left], nums[right]]);
                
                // Skip duplicates for left
                while (left < right && nums[left] === nums[left + 1]) {
                    left++;
                }
                // Skip duplicates for right
                while (left < right && nums[right] === nums[right - 1]) {
                    right--;
                }
                
                left++;
                right--;
            } else if (sum < 0) {
                left++; // Need larger sum
            } else {
                right--; // Need smaller sum
            }
        }
    }
    
    return result;
}

// Test
console.log(threeSum([-1,0,1,2,-1,-4])); // [[-1,-1,2],[-1,0,1]]
console.log(threeSum([0,1,1])); // []
console.log(threeSum([0,0,0])); // [[0,0,0]]
```

### Complexity Analysis
- **Time Complexity**: `O(n²)` - O(n log n) sort + O(n²) for nested loops
- **Space Complexity**: `O(1)` or `O(n)` depending on sort

## Key Takeaways

✅ Sort + two pointers for O(n²) solution  
✅ Skip duplicates to avoid duplicate triplets  
✅ Fix one element, use two pointers for other two  
✅ Early termination when nums[i] > 0  

**Pattern**: 3Sum → Sort + fix one + two pointers!
